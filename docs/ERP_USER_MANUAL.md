## 1. 系统概述

### 1.1 定位

Clawith ERP 是内置在多智能体协作平台中的轻量级企业资源管理系统，支持**客户、供应商、产品、物料、订单、生产、库存、财务、报表**全流程管理。

### 1.2 核心特点

- **模块化**：每个模块可独立启用/禁用（通过系统设置 → 模块配置）
- **多租户**：每个公司的数据完全隔离
- **智能体集成**：AI Agent 可直接操作 ERP 数据（创建订单、查询库存等）
- **蓝白色主题**：独立于 Clawith 主界面的专属 UI

### 1.3 访问方式

登录 Clawith 后，侧边栏点击 **ERP** 或直接访问 `https://你的域名/erp`

## 2. 系统设置

首次使用 ERP 前，建议先完成系统设置。

### 2.1 编码设置

路径：系统设置 → 编码设置

配置各业务单据的自动编号规则：**前缀 + N 位流水号**

| 单据类型 | 默认前缀 | 默认位数 | 示例 |
|---------|---------|---------|------|
| 客户编码 | K | 3 | K001, K002, K003 |
| 供应商编码 | G | 3 | G001, G002 |
| 产品编码 | P | 3 | P001, P002 |
| 物料编码 | M | 3 | M001, M002 |
| 销售订单编号 | SO | 4 | SO0001, SO0002 |
| 采购订单编号 | PO | 4 | PO0001, PO0002 |
| 出库单编号 | OUT | 4 | OUT0001 |
| 入库单编号 | IN | 4 | IN0001 |
| 调拨单编号 | TR | 4 | TR0001 |
| 财务记录编号 | FIN | 4 | FIN0001 |

### 2.2 分类管理

可为以下实体配置分类标签：

- **客户分类**：如 VIP客户、普通客户、渠道商
- **供应商分类**：如 原材料供应商、物流供应商
- **仓库分类**：如 成品仓、原料仓、退货仓
- **出库分类**：如 销售出库、生产领料、退货出库
- **入库分类**：如 采购入库、生产入库、退货入库

每个分类可设置一个**默认分类**，新建记录时自动选中。

### 2.3 生产状态

可自定义生产工单的状态流转，默认：开始 → 结束。可扩展为：备料中 → 生产中 → 质检中 → 已完成。

### 2.4 模块配置

控制各模块的启用/禁用：

| 模块 | 默认状态 | 说明 |
|------|---------|------|
| 客户管理 | 启用（不可关闭） | 基础模块 |
| 供应商管理 | 启用 | |
| 产品管理 | 启用 | 成品管理 |
| 物料管理 | 启用 | 原材料管理 |
| 库存管理 | 启用 | |
| 生产管理 | **禁用** | 按需开启 |
| 财务管理 | 启用 | |
| 收付款 | **禁用** | 按需开启 |

禁用的模块不会在侧边栏显示，不影响其他模块运行。

## 3. 基础数据管理

### 3.1 客户管理

路径：基础数据 → 客户管理

**字段**：客户编码（自动）、客户名称、简称、分类、业务员（默认当前用户）、地址、状态

**联系人**：每个客户可有多个联系人，支持设置默认联系人（显示在列表中）

**财务信息**：公司名称、统一社会信用代码、法人代表/电话、银行账户信息

**附件**：可上传任何文件或图片

**操作**：新建、编辑、删除（软删除）、启用/禁用切换

> 💡 新建客户后会自动切换到编辑模式，方便立即添加联系人和附件。

### 3.2 供应商管理

结构与客户管理完全对称，字段和操作相同。

### 3.3 产品管理

路径：基础数据 → 产品管理

产品是**销售侧**的成品，与客户、销售订单、销售收入关联。

**字段**：产品编码（自动）、名称、SKU、分类、单位、售价、库存量、最低库存（预警阈值）、状态

### 3.4 物料管理

路径：基础数据 → 物料管理

物料是**采购侧**的原材料，与供应商、采购订单、成本支出关联。

**字段**：物料编码（自动）、名称、SKU、分类、单位、成本价、库存量、最低库存、状态

### 3.5 BOM 管理（物料清单）

路径：基础数据 → BOM 管理

定义"生产 1 个产品需要哪些物料、各多少"。

**界面**：左侧产品列表，右侧该产品的 BOM 表格。

**字段**：物料名称、数量（生产 1 个产品需要多少）、单位

> 💡 BOM 是可选的。没有 BOM 的产品也可以生产（简易模式，只增加成品不扣物料）。

## 4. 销售业务

### 4.1 销售订单

路径：销售 → 销售订单

**订单状态流转**：
```
草稿(draft) → 已确认(confirmed) → 处理中(processing) → 已发货(shipped) → 已完成(completed)
                                                                 ↘ 已取消(cancelled)
```

**创建订单**：
1. 选择客户（可搜索筛选，只显示启用状态的客户）
2. 添加明细行：选择产品（可搜索） → 自动填充单价和单位 → 填写数量 → 自动计算金额
3. 可添加多行明细
4. 填写备注，提交

**确认订单**：
- 确认时如果开启了"自动库存扣减"（系统设置），会自动扣减产品库存
- 库存不足时整单拒绝，不会部分扣减

**查看订单**：点击订单行的眼睛图标，查看明细和状态流转按钮。

### 4.2 销售回款

路径：销售 → 销售回款

记录客户的付款信息，关联销售订单。

## 5. 采购业务

### 5.1 采购订单

路径：采购 → 采购订单

**订单状态流转**：
```
草稿(draft) → 已确认(confirmed) → 收货中(receiving) → 已完成(completed)
                                                     ↘ 已取消(cancelled)
```

**创建订单**：
1. 选择供应商（可搜索筛选）
2. 添加明细行：选择**物料**（不是产品）→ 自动填充成本价和单位 → 填写数量
3. 提交

**确认订单**：
- 确认时自动增加物料库存
- 写入入库记录

### 5.2 采购付款

路径：采购 → 采购付款

记录对供应商的付款信息，关联采购订单。

## 6. 生产管理

> ⚠️ 生产模块默认关闭，需在 系统设置 → 模块配置 中启用。

### 6.1 生产工单

路径：生产 → 生产工单

**生产模式**：
- **标准生产**：配置了 BOM 的产品，确认工单时自动扣减物料 + 增加成品
- **简易生产**：没有 BOM 的产品，确认工单时只增加成品

**创建工单**：
1. 选择要生产的产品（可搜索）
2. 填写生产数量
3. 如果该产品有 BOM，自动显示所需物料清单和总需求量
4. 提交

**确认工单**：
- 有 BOM：校验物料库存 → 扣减物料 → 增加成品
- 无 BOM：直接增加成品
- 写入出入库记录

---

## 7. 库存管理

### 7.1 库存总览

路径：库存 → 库存总览

显示所有产品和物料的当前库存，支持按类型筛选（产品/物料/全部）。

- 低于最低库存的行**红色高亮**
- 显示库存量、最低库存、单位

### 7.2 出入库操作

**入库**：选择产品/物料 → 选择仓库 → 填写数量和原因 → 确认

**出库**：同上，但会校验库存是否足够

**调拨**：选择产品/物料 → 源仓库 → 目标仓库 → 数量 → 确认

所有操作都会记录到出入库流水。

### 7.3 出入库记录

查看所有库存变动的流水记录，包括：
- 手动出入库
- 销售订单自动出库
- 采购订单自动入库
- 生产工单出入库
- 调拨记录

---

## 8. 财务管理

### 8.1 财务记录

路径：财务 → 财务记录

记录收入、支出、应收、应付等财务数据。

**字段**：类型（收入/支出/应收/应付）、分类、金额、关联订单、客户/供应商、支付方式、日期

**汇总**：页面顶部显示总收入、总支出、利润、应收、应付

---

## 9. 数据报表

路径：财务 → 数据报表

包含 5 张报表：

1. **销售报表**：按月/季/年的销售趋势折线图 + 数据表
2. **采购报表**：按月/季/年的采购趋势
3. **库存报表**：各产品/物料的库存量柱状图
4. **客户贡献排名**：Top 10 客户的销售额排名
5. **损益表**：收入、支出、利润的趋势分析

## 10. 业务闭环案例

### 案例：电子产品公司完整业务流程

假设一家公司生产蓝牙耳机，以下是完整的 ERP 使用流程：

#### 第一步：基础数据设置

1. **系统设置**：
   - 编码设置：客户 K+3位，供应商 G+3位，产品 P+3位，物料 M+3位
   - 客户分类：国内客户、海外客户
   - 供应商分类：芯片供应商、包装供应商
   - 仓库分类：成品仓、原料仓
   - 启用生产管理模块

2. **创建供应商**：
   - G001：深圳芯片科技（分类：芯片供应商）
   - G002：东莞包装厂（分类：包装供应商）

3. **创建物料**：
   - M001：蓝牙芯片（成本价 ¥15，单位：个）
   - M002：耳机外壳（成本价 ¥5，单位：个）
   - M003：包装盒（成本价 ¥2，单位：个）

4. **创建产品**：
   - P001：BT-100 蓝牙耳机（售价 ¥199，单位：个）

5. **配置 BOM**：
   - BT-100 = 蓝牙芯片 ×1 + 耳机外壳 ×1 + 包装盒 ×1

#### 第二步：采购入库

1. **创建采购订单**：
   - 供应商：深圳芯片科技
   - 物料：蓝牙芯片 ×1000，耳机外壳 ×1000
   - 确认订单 → 物料自动入库

2. **创建采购订单**：
   - 供应商：东莞包装厂
   - 物料：包装盒 ×1000
   - 确认订单 → 物料自动入库

3. **查看库存**：蓝牙芯片 1000、耳机外壳 1000、包装盒 1000

#### 第三步：生产

1. **创建生产工单**：
   - 产品：BT-100 蓝牙耳机
   - 数量：500
   - 系统显示所需物料：蓝牙芯片 500、耳机外壳 500、包装盒 500

2. **确认工单**：
   - 物料库存扣减：蓝牙芯片 500、耳机外壳 500、包装盒 500
   - 成品库存增加：BT-100 蓝牙耳机 +500
   - 查看库存：蓝牙芯片 500、耳机外壳 500、包装盒 500、BT-100 500

#### 第四步：销售

1. **创建客户**：
   - K001：京东商城（分类：国内客户）

2. **创建销售订单**：
   - 客户：京东商城
   - 产品：BT-100 蓝牙耳机 ×200
   - 金额自动计算：200 × ¥199 = ¥39,800

3. **确认订单**：
   - 产品库存扣减：BT-100 从 500 → 300
   - 写入出库记录

4. **记录回款**：
   - 关联销售订单，金额 ¥39,800，方式：银行转账

#### 第五步：查看报表

1. **销售报表**：本月销售 ¥39,800
2. **采购报表**：本月采购成本 ¥22,000（芯片 ¥15,000 + 外壳 ¥5,000 + 包装 ¥2,000）
3. **损益表**：收入 ¥39,800 - 成本 ¥22,000 = 毛利 ¥17,800
4. **库存报表**：BT-100 剩余 300、物料各剩余 500
5. **客户排名**：京东商城 ¥39,800 排名第一

### C. 常见问题

**Q: 为什么生产模块看不到？**
A: 生产模块默认关闭，需到 系统设置 → 模块配置 中启用。

**Q: 如何区分产品和物料？**
A: 产品是成品，用于销售；物料是原材料，用于采购和生产。两者库存独立管理。

**Q: 订单确认后能取消吗？**
A: 可以，但已扣减/增加的库存不会自动回滚，需要手动调整库存。

**Q: 编码规则能改吗？**
A: 可以，系统设置 → 编码设置 中修改前缀和位数。已有的编码不会自动更新。
